# VoidDE

> A macOS-inspired Linux desktop environment built entirely in Python + GTK4.
> 100% AI-generated code. Designed to run on Debian and spread to every distro.

![VoidDE Banner](docs/screenshot.png)

---

## What is VoidDE?

VoidDE is a full Linux desktop environment written in **Python** using **GTK4**.
It aims to bring the clean, minimal aesthetic of macOS to Linux — with a floating dock,
top menu bar, Spotlight-style launcher, and a polished dark GTK theme.

Every line of code in this project was generated by AI.

---

## Features

- macOS-style **top panel** — clock, battery, CPU, app name
- Floating **dock** with icon magnification
- **Spotlight launcher** — fuzzy search all installed apps
- Custom **GTK4 CSS theme** — dark mode, frosted glass, rounded corners
- **Window manager** built on python-xlib (X11)
- **Session manager** — autostart, watchdog, clean logout
- **Settings app** — appearance, dock, clock preferences
- Distro-agnostic — works on Debian, Arch, Fedora, openSUSE

## Install (one command)

```bash
git clone https://github.com/yourusername/voidde
cd voidde
bash install.sh
```

Then log out and select **VoidDE** at your login screen.

## Project structure

```
voidde/
├── core/
│   ├── wm.py          Window manager (X11)
│   └── session.py     Session manager
├── shell/
│   ├── panel.py       Top menu bar
│   ├── dock.py        App dock
│   └── launcher.py    Spotlight launcher
├── apps/
│   └── settings.py    System preferences
├── themes/
│   └── default/
│       └── gtk.css    GTK4 macOS theme
└── install.sh         One-command installer
```
